# Kitty Keybindings Reference (Linux/Windows Defaults)

This document summarizes the default keybindings for the Kitty terminal. The primary modifier on Linux and Windows is **CTRL+SHIFT**.

---

## Tab Management
| Action | Keybinding |
| :--- | :--- |
| **New Tab** | `CTRL+SHIFT+T` |
| **Close Tab** | `CTRL+SHIFT+Q` |
| **Next Tab** | `CTRL+SHIFT+RIGHT` |
| **Previous Tab** | `CTRL+SHIFT+LEFT` |
| **Move Tab Forward** | `CTRL+SHIFT+.` |
| **Move Tab Backward** | `CTRL+SHIFT+,` |
| **Set Tab Title** | `CTRL+SHIFT+ALT+T` |

---

## Window (Pane) Management
| Action | Keybinding |
| :--- | :--- |
| **New Window** | `CTRL+SHIFT+ENTER` |
| **Close Window** | `CTRL+SHIFT+W` |
| **Next Window** | `CTRL+SHIFT+]` |
| **Previous Window** | `CTRL+SHIFT+[` |
| **Move Window Forward** | `CTRL+SHIFT+F` |
| **Move Window Backward** | `CTRL+SHIFT+B` |
| **Move Window to Top** | `CTRL+SHIFT+BACKTICK` |
| **Enter Resize Mode** | `CTRL+SHIFT+R` |

---

## Layout & Appearance
| Action | Keybinding |
| :--- | :--- |
| **Next Layout** | `CTRL+SHIFT+L` |
| **Increase Font Size** | `CTRL+SHIFT+EQUALS` |
| **Decrease Font Size** | `CTRL+SHIFT+MINUS` |
| **Reset Font Size** | `CTRL+SHIFT+BACKSPACE` |
| **Toggle Fullscreen** | `CTRL+SHIFT+F11` |

---

## Clipboard & System
| Action | Keybinding |
| :--- | :--- |
| **Copy to Clipboard** | `CTRL+SHIFT+C` |
| **Paste from Clipboard** | `CTRL+SHIFT+V` |
| **Browse Scrollback** | `CTRL+SHIFT+H` |
| **Edit Configuration** | `CTRL+SHIFT+F2` |
| **Reload Configuration** | `CTRL+SHIFT+F5` |
| **Open Kitty Shell** | `CTRL+SHIFT+ESCAPE` |

---

> **Notes:**
> - These shortcuts use the default `kitty_mod` (Ctrl+Shift).
> - Use `CTRL+SHIFT+F1` to open the full official documentation within the terminal.
